# Python Sales Data Analysis

This project analyzes sales data using Python and the Pandas library to identify revenue trends and business insights.

## Objectives

- Calculate total revenue
- Analyze revenue by region
- Analyze revenue by product category
- Visualize results using charts

## Key Insights

- Total revenue: 4780
- North America generated the highest revenue
- Electronics dominate total sales
- Office supplies contribute the least revenue

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
